    Emacs Helm git-grep(1)

    Emacs fuzzy git-grep(1).

    Functions:

    M-x helm-git-grep

    Installation:

    In your emacs config:

    (add-to-list 'load-path "~/.emacs.d/load/path/helm-git-grep.el")
(require 'helm-git-grep)

    There is no need to setup load-path with add-to-list if you copy helm-git-grep.el to load-path directories.
